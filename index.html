###进程描述和控制

1、什么是指令跟踪？

2、进程的定义?

有几个定义：

  • 正在执行的程序。
  • 正在计算机上实行的程序实例。
  • 能分配给处理器并有处理器执行的实体。
  • 具有以下特征的活动单元：一组指令序列的执行、一个当前状态和相关的系统资源集。

3、通常有哪些事件会导致创建一个进程？

  • 在批处理环境中，相应作业提交时会创建进程；
  • 在交互环境中，当一个新用户试图登录时会创建进程；
  • 操作系统因提供一项服务而创建；
  • 由现在的进程派生。

5、抢占一个进程是什么意思？

一般来说，抢占这个术语被定义为收回一个进程正在使用的资源。在这种情况下，资源就是处理器本身。进程正在执行并可以继续执行，但是由于其他进程需要执行而被夺取处理器资源。

6、什么是交换，其目的是什么？什么是进程切换？

当内存中没有就绪状态的进程时，操作系统就是把被阻塞的进程换出到磁盘中的“挂起队列”的过程即为交换。

目的：当内存中的所有进程都处于阻塞态时，操作系统可以把其中的一个进程置于“挂起态”，并将它转移到磁盘中，内存中释放的空间可被调入的另一个进程使用。

进程切换的功能：在某一时刻，一个正在运行的进程被中断，操作系统指定另一个进程为运行态，并把控制权交给这个进程。

进程切换可以在操作系统从当前正在运行的进程中活动控制权的任何时刻发生。

7、如图所示七状态进程模型，为什么有两个阻塞状态？

因为系统为阻塞进程让出空间，所以有阻塞状态，阻塞挂起状态。

8、挂起进程的特点

  • 进程不能立即执行；
  • 进程可能是或不是正在等待一个事件。如果是，阻塞条件不依赖于挂起条件，阻塞事件的发生不会是进程立即被执行。
  • 为阻止进程执行，可以通过代理把这个进程置于挂起状态，代理可以是进程自己，也可以是父进程或操作系统。
  • 除非代理显式地命令系统进行状态转换，否则进程无法从这个状态中转移。

10、为什么需要两种模式（用户态和内核态）？

非特权态常称为用户态，这是因为用户程序通常在该模式下运行

特权态可称为系统态或内核态

使用两种模式的原因：它可以保护操作系统和重要的操作系统表（如进程控制块）不受用户程序的干涉。在内核态下，软件具有对处理器及所有指令、寄存器和内存的控制能力

程序状态字中有一位表示执行模式，这一位应某些事件的要求而改变。在典型情况下，当用户调用一个操作系统服务或 中断 触发 系统例程 的执行时，执行模式被设置成内核态；当从系统服务返回到用户进程时，执行模式被设置为用户态。

11、系统创建一个新进程所执行的步骤是什么？

一旦操作系统基于某种原因创建一个新进程：

  • 给新进程分配一个唯一的进程标识符；
  • 给进程分配空间；
  • 初始化进程控制块；
  • 设置正确的连接；
  • 创建或扩充其他数据结构。

12、中断和陷阱有什么区别？

1）中断与当前正在运行的进程无关的某种类型的外部事件相关（如完成一次IO操作）。控制首先转移给中断处理器，它做一些基本的辅助工作，然后转移到与已经发生的特定类型的中断相关的操作系统例程。

2）陷阱与当前正在运行的进程所产生的错误或异常条件相关。操作系统确定错误或异常条件是否是致命的。如果是，当前正在运行的进程被转换到退出态，并发生进程切换；如果不是，操作系统的动作取决于错误种类和操作系统的设计（试图恢复或通知用户），操作系统可能会进行一次进程切换或者继续执行当前正在运行的进程。

13、进程切换和模式切换的区别？

进程切换是，一个正在运行的进程被中断，操作系统指定另一个进程为运行态，并把控制权交给这个进程。进程切换可以在操作系统从当前正在运行的进程中获得控制权的任何时刻发生，由于进程之间不同状态的切换，需要重新分配各种资源，操作系统需要做更多的工作。

模式切换是，用户态和内核态之间的切换。因为他们的资源是共享的，所以效率高并且不改变正在运行的进程的状态。

###准备工作

  1. git安装:

    • 安装msysgit

  2. Node.js安装

###安装hexo
目前国内npm源有问题；所以键入如下代码即可安装成功:

npm install -g cnpm --registry=https://registry.npm.taobao.org
cnpm install hexo-cli -g

所有的npm用cnpm代替。

###创建blog文件夹
自己选择一个路径（如F:\blog）,在blog内鼠标点击右键，选择Git bash,并执行下面的命令：

hexo init

###安装依赖包

cnpm install

###本地查看

hexo generate
hexo server

本地博客已经搭建好，本地浏览器输入localhost:4000即可预览，别人无法浏览。

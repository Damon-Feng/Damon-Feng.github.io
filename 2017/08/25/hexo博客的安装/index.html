###准备工作

  1. git安装:

    • 安装msysgit

  2. Node.js安装

###安装hexo
目前国内npm源有问题；所以键入如下代码即可安装成功:

npm install -g cnpm --registry=https://registry.npm.taobao.org
cnpm install hexo-cli -g

所有的npm用cnpm代替。

###创建blog文件夹
自己选择一个路径（如F:\blog）,在blog内鼠标点击右键，选择Git bash,并执行下面的命令：

hexo init

###安装依赖包

cnpm install

###本地查看

hexo generate
hexo server

本地博客已经搭建好，本地浏览器输入localhost:4000即可预览，别人无法浏览。
